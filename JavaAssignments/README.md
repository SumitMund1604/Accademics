## How to Compile and Run

### For Regular Java Programs:
```bash
javac ProgramName.java
java ProgramName
```

### For Command-line Argument Programs:
```bash
javac ProgramName.java
java ProgramName arg1 arg2 arg3
```

### Example:
```bash
# Assignment 3, Question 4
javac Q4_CmdLineAddition.java
java Q4_CmdLineAddition 10 20

# Assignment 3, Question 7
javac Q7_CmdLineStudent.java
java Q7_CmdLineStudent 12345 "John Doe" CSE 85 90 78 88 92 86
```
